## 数据流

```
用户触发 / GitHub Actions Cron / WorkBuddy 定时任务
  │
  ├─→ fetch-news-ai.js
  │    │  1. 抓取 uapis.cn 热榜（知乎/微博/百度/头条/新浪/贴吧/V2EX/快手/少数派/IT之家/36氪/掘金/虎扑/抖音/B站 共15平台）
  │    │  2. 抓取额外信息源（AIbase/AI-Bot/猫目/GitHub Trending/Product Hunt/Hacker News）
  │    │  3. 抓取 新浪股市 API（上证/深证/创业板/恒生）
  │    │  4. 抓取 open-meteo 天气 API
  │    │  5. 调用 DeepSeek API 生成 10 版块新闻（含 AI 深度解析）+ TL;DR + 趋势
  │    └─→ 输出: data/{period}-latest.json
  │
  ├─→ gen-html.js
  │    │  · 双列紧凑目录导航
  │    │  · 卡片分隔线（边界感清晰）
  │    │  · 相对时间（刚刚/X分钟前/X小时前/X天前）
  │    │  · AI 深度解析（100% 覆盖，每条 100-200 字）
  │    └─→ 读取 JSON → 渲染 HTML 邮件 → 输出: data/{period}-email.html
  │
  └─→ mail-provider.js
       └─→ 读取 HTML → SMTP/API 发送 → 2973464622@qq.com
```

### 外部 API 调用量（单次运行）

| API | 调用次数 | 说明 |
|-----|---------|------|
| uapis.cn | 15次串行 | 热榜数据（300ms 间隔避免 429） |
| AIbase/AI-Bot/猫目 | 3次 | HTML 爬取 |
| GitHub/PH/HN | 3次 | API 获取 |
| DeepSeek | 5次 | AI 生成（4组版块 + 1次头条趋势） |
| 新浪财经 | 1次 | 股市指数（GBK 解码） |
| open-meteo | 1次 | 天气 |
| SMTP | 1次 | 发送邮件 |

## 核心脚本说明

### ai-provider.js

**职责**: 屏蔽不同 AI 提供商的 API 差异，提供统一 `.chat()` 接口。

**支持的协议**:
- **OpenAI 兼容** (默认): DeepSeek / MiniMax / OpenAI / 通义千问 / 智谱 / 豆包 / Kimi / Moonshot / Custom
- **Anthropic 原生**: Claude Sonnet / Haiku / Opus
- **Google Gemini 原生**: Gemini 1.5 / 2.0

**API 签名**:

```javascript
const ai = require('./ai-provider');

// 普通对话
const text = await ai.chat([
  { role: 'user', content: '你好' }
]);

// JSON 模式
const json = await ai.chatJSON([
  { role: 'user', content: '返回 JSON 数组' }
]);

// 列出所有支持的提供商
const providers = ai.listProviders();
```

**环境变量**: `AI_PROVIDER` / `AI_API_KEY` / `AI_MODEL` / `AI_BASE_URL` / `AI_TIMEOUT` / `AI_MAX_TOKENS` / `AI_TEMP`

**CLI 命令**:

```bash
node ai-provider.js --list    # 列出支持的提供商
node ai-provider.js --test    # 测试当前配置
node ai-provider.js --config  # 查看当前配置
```

### mail-provider.js

**职责**: 统一邮件发送层，支持 14+ 邮件服务商。

**支持的协议**:

**SMTP** (nodemailer):
- QQ 邮箱 (smtp.qq.com:465 SSL)
- 网易 163 (smtp.163.com:465 SSL)
- 网易 126 (smtp.126.com:465 SSL)
- 网易 yeah (smtp.yeah.net:465 SSL)
- Gmail (smtp.gmail.com:587 STARTTLS)
- Outlook/Hotmail (smtp-mail.outlook.com:587 STARTTLS)
- Yahoo (smtp.mail.yahoo.com:587 STARTTLS)
- 阿里云邮箱 (smtp.aliyun.com:465 SSL)
- QQ 企业邮箱 (smtp.exmail.qq.com:465 SSL)
- 自定义 SMTP

**HTTP API** (纯 REST，无需 SMTP 端口):
- Resend (https://api.resend.com/emails)
- SendGrid (https://api.sendgrid.com/v3/mail/send)
- Mailgun (https://api.mailgun.net/v3/{domain}/messages)
- Postmark (https://api.postmarkapp.com/email)
- Microsoft Graph (https://graph.microsoft.com/v1.0/me/sendMail)

**API 签名**:

```javascript
const mail = require('./mail-provider');

await mail.send({
  to: 'friend@qq.com',
  subject: '测试',
  html: '<h1>Hello</h1>',
});
```

**环境变量**: `MAIL_PROVIDER` / `MAIL_USER` / `MAIL_PASS` / `MAIL_FROM` / `MAIL_TO` / `MAIL_SMTP_HOST` / `MAIL_SMTP_PORT` / `MAIL_SMTP_SECURE` / `MAIL_API_KEY` / `MAILGUN_DOMAIN`

### fetch-news-ai.js

**职责**: 主流程编排——抓取数据 + 调用 AI + 降级 + 完整性检查。

**流程**:

1. **Run lock**: 30 分钟内同周期不重复运行
2. **热榜抓取**: 串行调 15 个 uapis.cn 接口（300ms 间隔，避免 429 限流）
3. **额外信息源**: AIbase/AI-Bot/猫目（HTML 爬取）+ GitHub Trending/Product Hunt/Hacker News（API）
4. **AI 生成**: 分 4 组（每组 ≤4 版块）并行调用，每条新闻必含 commentary（100-200 字深度解析）
5. **股市数据**: 新浪 API（GBK 编码，Buffer.concat 解码）
6. **天气**: open-meteo
7. **完整性检查**: GOOD(40+条) / DEGRADED(20-39条) / LOW(<20条)
8. **降级机制**: AI 失败 → 降级到纯热榜数据

**命令行**:

```bash
node fetch-news-ai.js --period=evening --output=data/evening-latest.json
ENABLED_SECTIONS=ai,finance,auto node fetch-news-ai.js --period=morning
```

### gen-html.js

**职责**: 读取 JSON → 渲染响应式 HTML 邮件。

**特性**:
- 4 时段不同渐变色头部
- 10 版块独立配色
- 📋 双列紧凑目录导航（固定入口 + 双列网格）
- ⚡ TL;DR 速览
- 📈 股市指数条
- ⭐ 星级评分
- 🕐 相对时间（刚刚/X分钟前/X小时前/X天前）
- 🧠 AI 深度解析（100% 覆盖，每条 100-200 字）
- 📝 卡片分隔线（边界感清晰）
- 🔗 原文链接
- 💬 每日一句

## GitHub Actions 工作流

### 触发方式

```yaml
# 1. 手动触发（可选 period）
workflow_dispatch:
  inputs:
    period:
      options: [auto, morning, noon, afternoon, evening]

# 2. 定时触发（UTC 时间）
schedule:
  - cron: '0 22 * * *'  # 北京时间 06:00
  - cron: '0 4 * * *'   # 北京时间 12:00
  - cron: '0 8 * * *'   # 北京时间 16:00
  - cron: '0 16 * * *'  # 北京时间 24:00

# 3. API 外部触发
repository_dispatch:
  types: [lumapost-send]
```

### 执行环境

- **Runner**: ubuntu-latest
- **Node.js**: 20
- **npm 缓存**: 基于 package-lock.json
- **Secrets**: 从 GitHub Secrets 读取
- **Artifacts**: 保留 7 天
- **失败处理**: AI 失败自动 fallback 到 fetch-cn-news.js

## 扩展开发

### 添加新 AI 提供商

在 `ai-provider.js` 的 `PROVIDERS` 对象添加：

```javascript
new_provider: {
  name: 'MyAI',
  baseUrl: 'https://api.myai.com/v1',
  defaultModel: 'my-model',
  envKey: 'MYAI_API_KEY',
  models: ['my-model-v1', 'my-model-v2'],
  docs: 'https://myai.com/docs',
  // 如果是 Anthropic 协议:
  // useAnthropicFormat: true,
  // 如果是 Google Gemini 协议:
  // useGoogleFormat: true,
}
```

### 添加新邮件服务商

在 `mail-provider.js` 的 `PROVIDERS` 对象添加：

```javascript
new_mail: {
  name: '新服务',
  type: 'smtp',  // 或 'api'
  host: 'smtp.new.com',  // SMTP 模式
```
